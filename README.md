# 该吃药啦

一个本地优先的 Android 服药提醒与药方管理应用。应用用于记录药方、服药时间和每日完成情况，并通过系统通知按时提醒。

## 功能

- 药方管理：新增、编辑、停用和删除药方，支持每日剂量与交替剂量。
- 服药提醒：按药方时间展示今日提醒，支持长按确认服药。
- 服用日历：查看每日服药状态，已到点计划会显示完成或未完成状态。
- 通知与时钟权限：支持系统通知权限、精准时钟权限和测试通知。
- 闹钟创建：可按药方时间并集选择性请求系统闹钟 App 创建闹钟。
- 数据备份：支持导入和导出本地 JSON 备份。
- 版本更新：通过 GitHub Release 检查并下载新版 APK。

## 数据与权限

应用数据保存在本机数据库中，不依赖账号体系。为了准时提醒，应用会使用通知权限和精准时钟权限；系统闹钟创建由手机系统闹钟 App 接管，Android 公共接口不支持写入闹钟有效期限。

## 构建

```powershell
.\gradlew.bat :app:assembleDebug
```

构建完成后，APK 位于：

```text
app/build/outputs/apk/debug/app-debug.apk
```
